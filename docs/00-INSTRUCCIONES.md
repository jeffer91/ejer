# Instrucciones de uso

## Probar la primera subpantalla con Live Server

Abre este archivo:

```text
src/pantallas/01-progreso/01-peso/prpe-index.html
```

Debe funcionar sola porque ya tiene:

```text
prpe-index.html
prpe.css
prpe-main.js
data/prpe-demo-data.json
```

## Pendiente para Electron

El archivo package.json no pudo subirse automáticamente desde la herramienta. Se debe crear manualmente en la raíz del proyecto con el contenido que se entregue en el chat.
