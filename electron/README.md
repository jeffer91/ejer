# Electron FitJeff

Esta carpeta contiene la base de escritorio de FitJeff.

## Desarrollo

```bash
npm run electron:dev
```

## Probar modo producción local

```bash
npm run electron:build
```

## Crear instalador Windows

```bash
npm run desktop:win
```

La app carga Vite en desarrollo y la carpeta `dist` cuando se construye para escritorio.
