# Control corporal

Este módulo agrupa la funcionalidad actual de FitJeff relacionada con peso, medidas, progreso e historial.

## Pantallas del módulo

- Estadísticas
- Registro
- Historial
- Inicio de configuración corporal

## Estado de migración

Bloque 2 crea la carpeta modular y conecta el router principal a este módulo.

Por seguridad, las pantallas internas todavía apuntan a la estructura anterior mediante archivos puente. En el siguiente bloque se pueden mover los archivos reales de cada pantalla a esta carpeta y limpiar las rutas antiguas sin romper la app.
