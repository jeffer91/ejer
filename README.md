# Fitness Jeff

App personal de vida fitness para registrar progreso, entrenamiento, ayuno, hidratación, recomendaciones con Gemini y voz.

## Comandos principales

```bash
npm install
npm start
```

## Preparación para APK

```bash
npm run cap:add:android
npm run cap:sync
npm run cap:open:android
```
