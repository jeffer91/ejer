# Fitness Jeff

App personal de bienestar y entrenamiento para registrar hábitos saludables, movimiento, hidratación, energía, descanso, recomendaciones con Gemini y voz.

## Enfoque de esta nueva versión

La app se reconstruye desde cero para que sea ordenada y fácil de corregir.

Reglas principales:

- Funciona por pantallas independientes.
- Cada subpantalla tendrá su propia carpeta, HTML, CSS, JS y datos demo.
- La base principal será Google Sheets.
- Firebase será respaldo alterno.
- Primero se guarda localmente y luego se sincroniza.
- Las recomendaciones de IA deben ser prudentes y enfocadas en hábitos saludables, no en extremos.

## Comandos previstos

```bash
npm install
npm start
```
